About Virto Connect
What is Virto Connect and how does it work?

Virto Connect is a customizable authentication component that allows users to easily sign up and sign in to applications built on the Virto Network.

Get started with these steps:

  • 1Enable the demo mode to connect to the server
  • 2Click the connect button
  • 3Sign up or sign in
  • 4Enjoy the Virto Network

Implementation Example

<!-- Add this to your HTML -->
<button onclick="virtoConnect.open()">Connect to Virto</button>

<!-- Configure the component with a custom server -->
<virto-connect 
  id="virtoConnect"
  form-type="login"
  server="https://your-server.com">
</virto-connect>

<!-- Import the script -->
<script type="module" async src="./virto-connect.js"></script>

<!-- Update server URL dynamically -->
<script>
  const virtoConnect = document.getElementById('virtoConnect');
  virtoConnect.serverUrl = 'https://new-server.com';
  // OR
  virtoConnect.setAttribute('server', 'https://new-server.com');
</script>
